# Remote Assist Demo

Este repositório é um esqueleto seguro para um projeto de demonstração com:
- servidor Python para receber comandos via TCP;
- app Android com serviço de acessibilidade (para uso legítimo em apps próprios);
- documentação e configuração base.

> Observação: este projeto não inclui lógica para cheats, bypass de segurança, ou automação em jogos.

## Estrutura
- `server/` — servidor TCP simples
- `android/` — projeto Android base
- `docs/` — documentação da arquitetura
- `config/` — arquivos de configuração

## Como usar
1. Configure o servidor Python.
2. Compile ou ajuste o app Android.
3. Use apenas em cenários autorizados e com consentimento.
