# Android Project Notes

Este diretório contém a estrutura base para um app Android que pode ser ajustado para:
- serviço de acessibilidade;
- comunicação com servidor TCP;
- testes em dispositivos próprios.

## Estrutura esperada
- `app/src/main/java/` — código Java/Kotlin do app
- `app/src/main/res/xml/` — configurações de acessibilidade e recursos
